# `mise config [flags] [subcommand]`

Manage config files

## Flags

### `--no-header`

Do not print table header

* [`mise config get [KEY] [-f --file <FILE>]`](/cli/config/get.md)
* [`mise config ls [--no-header]`](/cli/config/ls.md)
* [`mise config set [args] [flags]`](/cli/config/set.md)
