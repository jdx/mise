# `mise config`

**Usage**: `mise config [--no-header] <SUBCOMMAND>`

**Aliases**: `cfg`

Manage config files

## Flags

### `--no-header`

Do not print table header

* [`mise config generate [-o --output <OUTPUT>]`](/cli/config/generate.md)
* [`mise config get [-f --file <FILE>] [KEY]`](/cli/config/get.md)
