# `mise tasks`

**Usage**: `mise tasks [FLAGS] <SUBCOMMAND>`

**Aliases**: `t`

[experimental] Manage tasks

## Flags

### `--no-header`

Do not print table header

### `-x --extended`

Show all columns

### `--hidden`

Show hidden tasks

### `--sort <COLUMN>`

Sort by column. Default is name.

### `--sort-order <SORT_ORDER>`

Sort order. Default is asc.

### `-J --json`

Output in JSON format
