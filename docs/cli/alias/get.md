# `mise alias get`

**Usage**: `mise alias get <PLUGIN> <ALIAS>`

Show an alias for a plugin

This is the contents of an alias.&lt;PLUGIN> entry in ~/.config/mise/config.toml

## Arguments

### `<PLUGIN>`

The plugin to show the alias for

### `<ALIAS>`

The alias to show

Examples:

    $ mise alias get node lts-hydrogen
    20.0.0
