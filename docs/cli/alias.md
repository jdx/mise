# `mise alias`

**Usage**: `mise alias [-p --plugin <PLUGIN>] [--no-header] <SUBCOMMAND>`

**Aliases**: `a`

Manage aliases

## Flags

### `-p --plugin <PLUGIN>`

filter aliases by plugin

### `--no-header`

Don't show table header
