# `mise backends ls`

- **Usage**: `mise backends ls`
- **Aliases**: `list`
- **Source code**: [`src/cli/backends/ls.rs`](https://github.com/jdx/mise/blob/main/src/cli/backends/ls.rs)

List built-in backends

Examples:

```
$ mise backends ls
aqua
asdf
cargo
core
dotnet
gem
go
npm
pipx
spm
ubi
vfox
```
